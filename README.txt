How to publish on GitHub Pages (quick):
1) Create a new public repository on GitHub (e.g. davjamui-digital-library).
2) Upload all files and folders from this ZIP to the repository root.
   - On GitHub: Add file -> Upload files (or push via git)
3) In repository Settings -> Pages:
   - Source: Deploy from branch 'main' (or 'master'), folder: / (root).
4) Save. GitHub will provide a URL like https://<yourusername>.github.io/<repo-name>/
5) Visit the URL to see the site live.
